## Installation

### Backend (Spring Boot)
To set up the backend of the application, follow these steps:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/backend-repo.git
    ```

2. **Navigate to the backend directory:**
    ```bash
    cd backend-repo
    ```

3. **Install dependencies:**
    ```bash
    mvn install
    ```

4. **Start the server:**
    ```bash
    mvn spring-boot:run
    ```
